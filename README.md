## Setup
- Add `localforage` dependency using yarn or npm to your project
- Add `localforage` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    'localforage',

    // With options
    ['localforage', { /* module options */ }],
 ]
}
```
